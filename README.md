# Retail Store Database Management System (SQL Project)

This project is a **Retail Store Database Management System** built using SQL.  
It includes the complete database structure, sample data, and SQL relationships for managing categories, products, customers, orders, and order items.

The database is designed to help retail stores track:
- Products & categories  
- Customers  
- Orders  
- Items purchased in each order  

This project is perfect for learning **SQL, ER modeling, joins, constraints, and relational database design**.

---

## Project Features

### Well-designed database schema  
Covers complete retail workflow with proper **primary keys, foreign keys, and relationships**.

### Clean SQL Script  
- Creates tables  
- Adds constraints  
- Inserts sample data  
- Ready to run in MySQL, MariaDB, PostgreSQL, or SQL Server  

### Fully Normalized (3NF)  
No redundancy, clean relationships.
